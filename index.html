GeoServer is an open source server for sharing geospatial data.

Designed for interoperability, it publishes data from any major spatial data source using open standards.

Supported OGC Services

This product implements the Web Map Service, Revision 1.3, 1.1. This product implements Web Feature Service, Revision 2.0, 1.1, 1.0. This product implements Web Coverage Service, Revision 2.0, 1.1, 1.0. This product implements OGC API - Features, Revision 1.0. This product implements Web Map Tile Service, Revision 1.0. This product implements GeoTIFF, Revision 1.1. This product implements GPKG, Revision 1.2.

GeoServer implements Web Map Service, Revision 1.0. GeoServer implements Web Processing Service, Revision 1.0. GeoServer implements Catalog Services, Revsion 2.0. GeoServer implements community standards Tile Map Services and WMS-C.

GeoServer participates in the annual Free and Open Source Software for Geospatial conference.
